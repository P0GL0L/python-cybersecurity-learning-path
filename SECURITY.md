# Security Policy

## Supported Versions
This repository is a learning project. Only the latest `main` branch is supported.

## Reporting a Vulnerability
If you believe you have found a security issue:
- Do NOT open a public GitHub Issue with exploit details.
- Instead, open a GitHub Security Advisory (preferred), or contact the maintainer via GitHub profile.

## Scope
In-scope:
- Accidental secret exposure (keys, tokens, credentials)
- Unsafe example code that could cause harm if followed as-is
- Vulnerabilities in included scripts

Out-of-scope:
- Issues in third-party tools or dependencies not maintained in this repository

## Response Targets (Best Effort)
- Acknowledgement: within 7 days
- Fix or mitigation: as soon as practical
