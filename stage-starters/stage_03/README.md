# Stage 3 - Files, Automation, and Robust CLI Programs

## Goal
Log Analyzer: parse a log file, produce a summary report, export JSON; Backup/Sync helper: copy files by pattern with a dry-run mode

## Acceptance Criteria
- Program supports --help, --input, --output, --verbose flags; Program exits with meaningful status codes

## How to Run
```bash
python app/main.py --help
```
